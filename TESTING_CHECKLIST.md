# Wild Motion — Testing Checklist

---

## 🇬🇧 SIMPLE ENGLISH VERSION

**Before you start, please have open in Figma:**
- A frame with 3–4 layers (rectangles, text, shapes)
- One audio file (MP3 or WAV) ready to use

---

### 1. Plugin Opens
- [ ] Plugin opens without any errors
- [ ] You can drag the bottom edge to make the plugin taller or shorter
- [ ] Close and reopen the plugin — everything is still there

---

### 2. Home Screen
- [ ] You can create a new project
- [ ] You can open a project
- [ ] You can rename a project
- [ ] You can delete a project (only if there is more than one)

---

### 3. Import Layers
- [ ] Select 2–3 layers in Figma, click **Import** — they appear in the timeline
- [ ] The layer names match what you see in Figma
- [ ] **Text layers** — select a text layer, import it, add a scale keyframe. Does it scale correctly without jumping?

---

### 4. Timeline — Basic
- [ ] Click on the timeline ruler — the playhead moves to that spot
- [ ] Click the ◀| button (top left of timeline) — playhead jumps to the **previous** keyframe
- [ ] Click the |▶ button — playhead jumps to the **next** keyframe
- [ ] Scroll down in the timeline with many layers — the layer names and keyframe tracks scroll together (they stay aligned)

---

### 5. Keyframes
- [ ] Click the diamond icon on a property row to **add** a keyframe
- [ ] Right-click a keyframe to **delete** it
- [ ] Click one keyframe, then **Shift+click** more keyframes across different layers — they all highlight
- [ ] Press **Delete** (or Backspace) — all selected keyframes are deleted (not just one)
- [ ] Press **Cmd+Z** — the deleted keyframes come back
- [ ] Press **Cmd+Shift+Z** — redo works

---

### 6. Curve Editor
- [ ] Click a keyframe — a **Curve Editor panel** opens on the right side of the plugin
- [ ] You can change the easing curve (drag the handles)
- [ ] You can pick a preset curve (Linear, Ease In, etc.)
- [ ] Click the ✕ button — the Curve Editor closes
- [ ] Click the 🗑 button — the keyframe is deleted
- [ ] Open the **Presets panel** while a keyframe is selected — both panels appear side by side

---

### 7. Playback & Audio
- [ ] Press **Space** — animation plays
- [ ] Press **Space** again — it stops
- [ ] Import an audio file. Press play — music starts with the animation
- [ ] **Scrub** (drag) the playhead while playing — music stops when you touch the ruler ✅
- [ ] Play the animation all the way to the end — it loops back to the start and **music restarts** ✅
- [ ] Stop the animation — layers go back to their original positions in Figma

---

### 8. Export

#### Lottie JSON
- [ ] Click **Export → Lottie**
- [ ] Open the downloaded `.json` file in [lottiefiles.com/preview](https://lottiefiles.com/preview)
- [ ] The animation **plays and is visible** (not blank/invisible) ✅

#### GIF
- [ ] Click **Export → GIF**
- [ ] The progress bar fills up and a `.gif` file downloads
- [ ] Open the GIF — it plays correctly

#### MP4
- [ ] Click **Export → MP4** (Chrome or Edge only)
- [ ] The progress bar fills up and a `.mp4` file downloads
- [ ] Open the MP4 — it plays correctly

---

### 9. Anything Else?
Please write down anything that:
- Looks wrong or broken
- Feels confusing or hard to use
- You think could be better

Thank you! 🙏

## 🇮🇩 VERSI BAHASA INDONESIA

**Sebelum mulai, tolong buka di Figma:**
- Sebuah frame dengan 3–4 layer (kotak, teks, atau shape)
- Satu file audio (MP3 atau WAV) yang siap dipakai

---

### 1. Plugin Terbuka
- [ ] Plugin terbuka tanpa error
- [ ] Bisa drag bagian bawah plugin untuk mengubah tingginya
- [ ] Tutup dan buka lagi plugin — semua data masih ada

---

### 2. Halaman Utama (Home)
- [ ] Bisa buat project baru
- [ ] Bisa buka project
- [ ] Bisa ganti nama project
- [ ] Bisa hapus project (hanya jika ada lebih dari satu)

---

### 3. Import Layer
- [ ] Pilih 2–3 layer di Figma, klik **Import** — layer muncul di timeline
- [ ] Nama layer sama dengan yang ada di Figma
- [ ] **Layer teks** — pilih layer teks, import, tambahkan keyframe skala. Apakah skalanya berubah dengan benar tanpa melompat/bergeser?

---

### 4. Timeline — Dasar
- [ ] Klik ruler timeline — playhead bergerak ke posisi itu
- [ ] Klik tombol ◀| (kiri atas timeline) — playhead loncat ke keyframe **sebelumnya**
- [ ] Klik tombol |▶ — playhead loncat ke keyframe **berikutnya**
- [ ] Scroll ke bawah di timeline dengan banyak layer — nama layer dan track keyframe ikut scroll bersama (tidak bergeser sendiri-sendiri)

---

### 5. Keyframe
- [ ] Klik ikon berlian di baris properti untuk **tambah** keyframe
- [ ] Klik kanan keyframe untuk **hapus** satu keyframe
- [ ] Klik satu keyframe, lalu **Shift+klik** keyframe lain di layer yang berbeda — semuanya ter-highlight
- [ ] Tekan **Delete** (atau Backspace) — semua keyframe yang dipilih terhapus (bukan hanya satu) ✅
- [ ] Tekan **Cmd+Z** — keyframe yang dihapus kembali lagi
- [ ] Tekan **Cmd+Shift+Z** — redo berhasil

---

### 6. Curve Editor
- [ ] Klik satu keyframe — panel **Curve Editor** muncul di sisi kanan plugin
- [ ] Bisa ubah kurva easing (drag handle-nya)
- [ ] Bisa pilih preset kurva (Linear, Ease In, dll.)
- [ ] Klik tombol ✕ — Curve Editor tertutup
- [ ] Klik tombol 🗑 — keyframe terhapus
- [ ] Buka **panel Presets** saat keyframe dipilih — kedua panel muncul berdampingan

---

### 7. Playback & Audio
- [ ] Tekan **Space** — animasi berjalan
- [ ] Tekan **Space** lagi — animasi berhenti
- [ ] Import file audio. Tekan play — musik mulai bersama animasi
- [ ] **Scrub** (drag) playhead saat animasi berjalan — musik berhenti saat kamu menyentuh ruler ✅
- [ ] Mainkan animasi sampai selesai — animasi loop ke awal dan **musik mulai lagi dari awal** ✅
- [ ] Stop animasi — layer kembali ke posisi aslinya di Figma

---

### 8. Export

#### Lottie JSON
- [ ] Klik **Export → Lottie**
- [ ] Buka file `.json` yang diunduh di [lottiefiles.com/preview](https://lottiefiles.com/preview)
- [ ] Animasi **terlihat dan berjalan** (tidak kosong/tidak terlihat) ✅

#### GIF
- [ ] Klik **Export → GIF**
- [ ] Progress bar terisi dan file `.gif` terunduh
- [ ] Buka GIF — animasi berjalan dengan benar

#### MP4
- [ ] Klik **Export → MP4** (hanya di Chrome atau Edge)
- [ ] Progress bar terisi dan file `.mp4` terunduh
- [ ] Buka MP4 — video berjalan dengan benar

---

### 9. Catatan Lain
Tolong tulis kalau ada yang:
- Kelihatan salah atau rusak
- Membingungkan atau sulit dipakai
- Menurutmu bisa dibuat lebih baik

Terima kasih banyak! 🙏
